Base Frontend Task 12 — Модальные окна

## Live Demo

https://ereminaolesya.github.io/basefrontend-task12/

## Описание

Реализация HTML+CSS+JavaScript проект доступного модального окна.
Задача взята из учебного практического курса.

## Требования к задаче
Функциональные требования:
открытие модального окна по кнопке на экране
закрытие модального окна по кнопке-крестику и по overlay’у модального окна

Нефункциональные требования:
модальное окно должно скрываться/отображаться через анимацию fade-in/fade-out длительностью 300 мс

## Stack

-HTML
-CSS
-JavaScript
